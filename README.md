# 2026 深度強化學習期末專題

## 基於 LoRA 與強化學習之 LLM 推理優化

### 📋 專題概述

本專題致力於優化大型語言模型 (LLM) 的推理性能，通過結合 LoRA (Low-Rank Adaptation) 技術與強化學習方法，實現更高效的模型推理。

## 🎯 主要特性

- **LoRA 集成**: 採用低秩適應技術實現高效的模型微調
- **強化學習優化**: 利用強化學習算法優化推理策略
- **性能提升**: 在保證模型準確度的前提下，顯著降低計算開銷
- **可擴展性**: 支持多種 LLM 架構
